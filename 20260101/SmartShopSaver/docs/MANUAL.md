### 📍 連接 Gmail

讓系統自動從您的購物郵件中識別消費記錄。

#### 使用方式

```
連接 Gmail
綁定 Gmail
Gmail 授權
```

#### 授權步驟

1. 發送「連接 Gmail」
2. 點擊回覆中的授權連結
3. 在 Google 頁面選擇帳號
4. 點擊「允許」授權
5. 授權成功後自動返回

• 僅讀取購物郵件
• 不會修改或刪除內容

## 常見問題

### ❓ 價格追蹤多久更新一次？

系統每 **6 小時**自動檢查追蹤商品的價格。當價格達到或低於您設定的目標價格時，會**立即**發送 LINE 通知給您。

---

### ❓ Gmail 授權安全嗎？

**完全安全**。我們使用 Google 官方的 OAuth 2.0 授權機制：

- ✅ 只讀取郵件，不會修改或刪除
- ✅ 只分析購物相關郵件
- ✅ 不儲存原始郵件內容
- ✅ 您可以隨時取消授權

**取消授權方式：**
1. 前往 [Google 帳戶設定](https://myaccount.google.com/permissions)
2. 找到「SmartShopSaver」
3. 點擊「移除存取權」

---

### ❓ 記帳資料會保存多久？

- **記帳記錄**：永久保存
- **價格追蹤**：持續追蹤直到您移除
- **購物郵件記錄**：保存最近 12 個月

---

### ❓ AI 推薦功能需要付費嗎？

**不需要**。基本功能完全免費，AI 推薦和評價分析功能由服務提供者負擔 OpenAI API 費用，用戶無需付費。

---

### ❓ 為什麼有時候 AI 回應比較慢？

AI 功能需要連接 OpenAI API 處理，通常需要 3-10 秒。如果超過 30 秒沒有回應，系統會提供備用的基本回應。

---

### ❓ 可以刪除記帳記錄嗎？

目前暫不支援單筆刪除功能。如需修正錯誤記錄，可以記一筆負數金額來沖銷：

```
記帳 沖銷午餐 -150
```

---

### ❓ 遇到問題怎麼辦？

1. **輸入「幫助」** 查看使用說明
2. **檢查指令格式** 是否正確
3. **稍後再試** 可能是暫時性問題
4. **回報問題** 透過 GitHub Issues
